# Kafka Interview Notes

Q: What is Kafka?

A:
Kafka is a distributed event streaming platform
used for real-time data pipelines and streaming applications.

------------------------------------------------

Q: What is a Producer?

A:
Producer publishes messages to Kafka topics.

------------------------------------------------

Q: What is a Consumer?

A:
Consumer reads messages from Kafka topics.

------------------------------------------------

Q: What is a Topic?

A:
Logical category or stream of messages.

------------------------------------------------

Q: What is a Partition?

A:
A topic subdivision used for parallelism
and scalability.

------------------------------------------------

Q: What is an Offset?

A:
Unique position of a message within a partition.

------------------------------------------------

Q: Why is Kafka popular?

A:
- High throughput
- Scalability
- Durability
- Fault tolerance
- Event replay capability

------------------------------------------------

Q: Explain Kafka Flow.

A:

Producer
   ↓
Topic
   ↓
Consumer
